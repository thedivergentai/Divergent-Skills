# Instagram Creator Marketplace API

Instagram's creator marketplace is where you can discover and evaluate Instagram creators for [partnership ads](https://l.facebook.com/l.php?u=https%3A%2F%2Fhelp.instagram.com%2F292748974937716&h=AUDERJKqKsVFUVhVIn43Ku9Pf_-Es-VdO-Ryhe5NYuBMiBVs8VJZKX4nPOIoIqUzDOM-VoI3SCi83DydMa1BEOP18V3PqUU6KkM9Wva6IJlId1t2chYMO1kiUieZJ9P9G1HO3xktL9S5DCNw). The API offers personalized creator recommendations and search using authenticated first-party data to help your brand find the right creators for your partnership ad campaigns. You can evaluate creators for partnership ads using authenticated, real-time first-party data.
