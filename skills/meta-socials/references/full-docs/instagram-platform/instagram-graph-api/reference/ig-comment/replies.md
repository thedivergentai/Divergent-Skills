# IG Comment Replies

Represents a collection of [IG Comments](instagram-api/reference/ig-comment.md) on an [IG Comment](instagram-api/reference/ig-comment.md).

To create an [IG Comment](instagram-api/reference/ig-comment.md) on an [IG Media](instagram-api/reference/ig-media.md) object, use the [`POST /{ig-media-id}/comments`](instagram-api/reference/ig-media/comments.md) endpoint instead.
