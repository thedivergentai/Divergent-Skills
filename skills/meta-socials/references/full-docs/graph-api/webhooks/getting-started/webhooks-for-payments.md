# Webhooks for Payments

Webhooks for [Payments](games_payments.md) allows you to receive real-time notifications whenever a payment status has changed. Webhooks for Payments works a little differently than other Webhooks so please use the [Webhooks for Payments docs](games_payments/webhooks.md) when setting up this type of Webhook.
