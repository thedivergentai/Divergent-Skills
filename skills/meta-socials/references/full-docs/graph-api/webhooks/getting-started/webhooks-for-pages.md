# Webhooks for Pages

Webhooks for [Pages](pages.md) can send you real-time notifications of changes to your Pages. For example, you can receive real-time updates whenever users post to your feed, comment on a post, or like your posts.

To set up a Page Webhook:

1. [Set up your endpoint and configure the Webhooks product](#set-up-endpoint-and-product).
2. [Install your app](#install-app) using your Facebook page.
