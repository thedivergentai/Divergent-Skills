# Webhooks from Meta

Webhooks allows you to receive real-time HTTP notifications of changes to specific objects in the Meta social graph. For example, we could send you a notification when any of your app Users change their email address or whenever they comment on your Facebook Page. This prevents you from having to query the Graph API for changes to objects that may or may not have happened, and helps you avoid reaching your [rate limit](graph-api/advanced/rate-limiting.md).

[Webhooks for Payments](games_payments/webhooks.md) and [Webhooks for Messenger](messenger-platform/webhook.md) have slightly differently configuration steps. If you are setting up a Webhook for either of these products, please refer to their respective documents for setup instructions.
