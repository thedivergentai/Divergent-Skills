# June 30, 2021 - Graph API

Source: https://developers.facebook.com/docs/graph-api/changelog/ooc-changes/jun-30-2021

## Instagram Basic Display API

#### Private Account Access Tokens

*Applies to all versions.*

[Long-lived access tokens](instagram-basic-display-api/guides/long-lived-access-tokens.md) for private Instagram accounts can now be [refreshed](instagram-basic-display-api/guides/long-lived-access-tokens.md#refresh-a-long-lived-token). In addition, [permissions](instagram-basic-display-api/overview.md#permissions) granted to apps by app users with private accounts are now valid for 90 days.
