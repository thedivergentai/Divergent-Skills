# Secure Graph API Calls

Almost every Graph API call requires an [access token](facebook-login/access-tokens.md). Malicious developers can steal access tokens and use them to send spam from your app. Meta has automated systems to detect this, but you can help us secure your app. This document covers some of the ways you can improve security in your app.
