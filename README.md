# Divergent Skills

Agentic skillset library by [Divergent AI](https://github.com/divergent-ai) — skills used consistently or found lacking in the current skills ecosystem.

Please feel free to leave issues and comments. That's the readme... Kinda.

## Skills

| Skill | Description |
|-------|-------------|
| [weasyprint](skills/weasyprint/) | Master skill for HTML/CSS → PDF with WeasyPrint, official CourtBouillon examples, and scraped docs |

## Install

```bash
npx skills add https://github.com/divergent-ai/divergent-skills --skill weasyprint
```

Replace the repo URL with this repository once published.

## Audits

Skills are audited against [skill-judge](https://github.com/softaworks/agent-toolkit/tree/main/skills/skill-judge). Reports and naming rules: [`.audit/README.md`](.audit/README.md). WeasyPrint history: [`.audit/weasyprint--index.md`](.audit/weasyprint--index.md).
